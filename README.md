# 8-bit ALU with Control Unit and Testbench

## Overview
This project implements an 8-bit Arithmetic Logic Unit (ALU) using Verilog HDL along with a testbench for simulation and verification.

## Features
- 8-bit operations
- Arithmetic: Addition, Subtraction
- Logic: AND, OR, XOR, NOT
- Shift operations
- Fully verified using testbench

## Files
- alu_8bit.v → Main ALU design
- testbench.v → Simulation file

## Tools
- Verilog HDL
- ModelSim / Xilinx
